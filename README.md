## GitHub Viewer

Get information from a specific user like email, location, followers, bio and repositories.

## ⚙️ Requirements

👨‍💻 Node.js<br />
🐈 Yarn

## 🚀 Run the app

Run `yarn` and then `yarn start` to run the application on port 3000.

## 💻 Test it!

Type the username in the form field.<br />
